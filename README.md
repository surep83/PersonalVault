# Personal Vault Full MVP v4

- Multi-type: passwords, documents, contacts, events, custom
- Folder and tag organization
- Search bar UI
- Document preview: PDF, TXT, Images
- Automatic thumbnail generation for PDFs/images
- Background auto-sync with Google Drive
- Encrypted with Argon2 + VaultCrypto
- SQLite offline storage
- Mobile-first Flutter UI

Open in VS Code, run 'flutter pub get', and launch on device.
